# 客户与组织 - 前端 Demo 说明

## Demo 范围
- 组织管理
  - 组织列表
  - 组织详情与编辑
- 客户管理
  - 客户列表
  - 客户详情与编辑
  - 客户关系维护（品牌商、零售商、供应商、部门、头衔）

## 领域模型映射
- Organization 聚合 -> 组织管理页面
- Customer 聚合 -> 客户管理页面
- OrganizationRelationship 与客户关系聚合 -> 关系管理页签
- OrganizationTag -> 组织标签与关系类型选择
- OrganizationStatus 与 CustomerStatus -> 列表状态标签

## 交互说明
- 左侧导航仅保留组织列表与客户列表
- 组织列表支持关键字与标签筛选
- 组织详情采用基础信息与联系人信息双页签
- 组织基础信息字段包含编码、名称、全称、状态、标签、是否有效、备注
- 组织联系人信息字段包含联系人姓名、电话、邮箱、类型、备注
- 客户新建含多页签：基础信息、品牌商、零售商、供应商、部门、头衔关系
- 客户基础信息字段包含客户编码、名称、全称、组织ID（只读）、状态、是否有效、行业、联系人姓名/电话/邮箱/角色
- 关系页签新增关系时使用下拉选择组织，按当前页签标签过滤组织列表
- 关系页签支持新增与删除关系
