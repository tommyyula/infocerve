# 自动化测试指南

> 使用 Playwright MCP 工具进行页面功能测试

---

## 测试工具

| 工具 | 用途 |
|------|------|
| `browser_navigate` | 导航到指定 URL |
| `browser_snapshot` | 获取页面快照（推荐） |
| `browser_click` | 点击元素 |
| `browser_type` | 输入文本 |
| `browser_fill_form` | 批量填写表单 |

---

## 测试流程

1. **登录** → 导航到登录页，填写凭据，提交
2. **验证登录** → 检查页面跳转和用户信息
3. **功能测试** → 逐个测试 CRUD 操作
4. **记录问题** → 发现 BUG 输出到 `.kiro/bugs/`

---

## BUG 报告格式

```
摘要: 简洁描述问题

环境:
URL: 问题页面地址
系统: 所属系统/模块

复现步骤
1. 步骤一
2. 步骤二
...

实际结果
观察到的实际行为

期望结果
期望的正确行为

备注
补充说明（可选）
```
